Branded social link row — icon, network name, handle, per-brand hover glow.

```jsx
<SocialLink network="LinkedIn" handle="abhishek-singh" color="#0A66C2"
  href="https://www.linkedin.com/in/abhishek-singh-3a54a5294/"
  icon="https://cdn.simpleicons.org/linkedin/0A66C2" />
<SocialLink network="GitHub" handle="@Abhishek5467" color="#ffffff"
  icon="https://cdn.simpleicons.org/github/white" href="…" />
```

`icon` takes a simple-icons URL or an inline SVG. `color` drives the hover glow to match each brand.
