Infinite horizontal ticker for skill / tech keywords. Pauses on hover, edges fade out.

```jsx
<Marquee items={["Computer Vision","Silicon Photonics","PyTorch","Deep Learning","Cloud","Embedded AI"]} />
<Marquee items={["IIT PATNA","RESEARCH","AI"]} reverse speed={40} size="2rem" />
```

Items are duplicated for a seamless loop; `reverse` and `speed` tune direction/rate.
