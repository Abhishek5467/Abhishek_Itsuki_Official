Standard section opener: mono eyebrow, display title, description.

```jsx
<SectionHeading
  index="02" eyebrow="Research"
  title={<>Detecting the <em>anomalous</em> frame</>}
  description="Unsupervised video anomaly detection with dataset-adaptive scoring."
/>
```

Wrap part of `title` in `<em>` to get the photon→violet gradient highlight. `align="center"` for centered hero sections.
