Compact square or round control for social icons and toolbar actions.

```jsx
<IconButton href="https://github.com/Abhishek5467" label="GitHub" round>
  <img src="https://cdn.simpleicons.org/github/white" alt="" />
</IconButton>
```

Pass an `<svg>` or `<img>` child. `round` for circular, `ghost` for transparent, sizes `sm|md|lg`. Always set `label` for accessibility.
