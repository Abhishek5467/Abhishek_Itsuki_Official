Mono-caps status pill for domains, states and labels.

```jsx
<Badge tone="photon" dot>Online</Badge>
<Badge tone="optic">Photonics</Badge>
<Badge tone="signal">Anomaly</Badge>
<Badge tone="neutral">PyTorch</Badge>
```

Tones map to the accent palette: `photon` (cyan), `optic` (violet), `resonance` (amber), `signal` (red), `lime` (green), `neutral`, `solid`. Set `dot` for a glowing live-status indicator.
