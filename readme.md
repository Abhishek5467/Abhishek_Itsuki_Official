## About

Abhishek Singh is a B.Tech Electronics and Communication Engineering student at IIT Patna with interests spanning software engineering, competitive programming, artificial intelligence, photonics, embedded systems, and applied mathematics.

His work combines research, engineering, and problem solving. He has developed projects in video anomaly detection, integrated photonics simulation, embedded healthcare devices, full-stack web applications, and creative technology. His research interests include machine learning, computer vision, computational electromagnetics, and silicon photonics.

Alongside academics and research, he is an active competitive programmer with experience across Codeforces, CodeChef, LeetCode, and GeeksforGeeks, having solved more than 1000 programming problems across multiple platforms.

His technical work ranges from designing deep-learning systems for anomaly detection to simulating photonic devices using FDTD methods, building embedded systems with ESP32, and developing modern web applications using React, Node.js, and cloud technologies.
